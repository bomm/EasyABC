# EasyABC

An open source ABC editor for Windows, OSX and Linux. It is published under the GNU Public License.

## Attention

This is a fork of https://github.com/jwdj/EasyABC.

The purpose of this fork is to provide bugfixes or improvements for the original repository.

Currently, the following work is in progress:

- Add files for creating a Debian package, also suitable for other Debian-based distributions like Ubuntu or Mint.
- Update the German translation

## Problem reporting

Please report here only problems that are related to the Debian package or the German translation.

Examples for possible problems related to the Debian package:

- missing dependencies, e.g. trying to run EasyABC fails with an error message about a missing Python package
- superfluous dependencies
- wrong dependencies
- wrong desktop integration (**desktop** menus, categories)
- problems in the installation scripts
- problems in the scripts for building the Debian package
- non-compliance to rules or style guides for Debian packages

Problems in EasyABC itself should be reported here in the upstream repository https://github.com/jwdj/EasyABC. 

Examples for problems *NOT* to be reported here:

- warnings when running EasyABC
- wrong display of the ABC source code or the music score
- problems playing the music
- problems in exporting or printing the music score
- errors in the menus or toolboxes
